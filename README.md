# Dra. Paula Pires - Psychologist Landing Page 🧠

> A responsive landing page for a clinical psychologist. This is my very first website coded entirely from scratch using HTML, CSS, and JavaScript.

## 🚀 Technologies Used

* **HTML5:** Semantic structure.
* **CSS3:** Custom styling, Flexbox, responsive design, and CSS variables (`:root`).
* **JavaScript (Vanilla):** Interactive DOM manipulation (Mobile menu toggle).
* **Remix Icon:** Open-source icons for the UI.
* **Google Fonts:** Custom typography (Italiana & Montserrat).

## ✨ Features

* Fully responsive layout (adapts to desktop and mobile screens).
* Interactive mobile navigation menu (Hamburger icon that toggles to a 'Close' icon).
* Clean, modern, and professional UI tailored for a healthcare professional.

## 🧠 What I Learned

As my first hand-coded project, this built a strong foundation in web development. I learned:
* The core relationship between HTML, CSS, and JavaScript.
* How to properly structure a project directory (`assets/`, `index.html`, `styles.css`, `main.js`).
* How to use CSS Variables (`:root`) to maintain a consistent color theme.
* Basic DOM manipulation in JS (`getElementById`, `classList.toggle`, `addEventListener`).
* How to manage layout positioning and `z-index` for fixed navigation bars.
